# sistema-de-ticket

Este proyecto es una aplicación estática de seguimiento de incidencias operativas que puede ejecutarse directamente desde GitHub Pages.

## Ejecución desde GitHub

1. Copia el repositorio en GitHub.
2. En el repositorio, ve a `Settings` > `Pages`.
3. Selecciona la rama `main` (o la rama que uses) y la carpeta `/ (root)`.
4. Guarda la configuración.
5. Accede a la URL de GitHub Pages que te proporcione GitHub.

## Archivos clave

- `index.html` — página principal lista para GitHub Pages.
- `SISTEMA DE TICKETS.html` — archivo original.
- `.nojekyll` — evita que GitHub Pages ignore archivos con rutas o nombres especiales.

## Uso local

1. Asegúrate de tener Node.js instalado.
2. Abre una terminal en la carpeta del proyecto.
3. Ejecuta `npm start`.
4. Abre `http://localhost:8080` en tu navegador.

La aplicación se sirve como una URL local y funciona igual que en GitHub Pages.
